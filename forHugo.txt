récap GIT

tu pull pour récup les majs du projet avant de commencer un truc sur le projet
git pull

une fois que tu as pull le projet tu vas créer une branch
git branch *leNomDeTaBranche*
Tu change pour aller sur la branch avec un
git checkout *leNomDeTaBranche*

Une fois que tu as fais UNE fonctionnalité tu vas venir ajouter tes fichiers
tu peux faire un 
git status
Pour lister les fichiers que tu as de modifier depuis le DERNIER commit de ton ordi et pas du DEPOTS

Pour ajouter les fichiers a la liste des choses à versioner tu fais ça (tu peux enchainer et faire plusieur git add de suite)
    git add *leNomDuFichier1QueTuAsEdite* *fichier2* *fichierN*

tu refais un derneir git status pour s'assurer que tu as bien mis les éléments que tu voulais

Maintenant tu vas pouvoir valider les changements avec un 
    git commit -m "*ton message*"       # eviter les message trop long

et pour finir tu fais un 
    git pull 
puis un 
    git push          # si tu as un
